# 前言
ijkplayer是一款跨平台的播放器，支持Android与iOS端，核心部分基于ffmpeg，</br>
支持Android的mediacodec硬解与iOS的videotoolbox硬解，视频图像采用OpenGL进行渲染。</br>
许多主流播放器都使用ijkplayer作为播放方案。</br>
